# mkv_merger.py

A Python script with the goal to merge MKV files within a directory with their corresponding audio, subtitle, chapter, attachment files. Imagine a TV series where you want to add a bunch of subtitle files and audio files to them in batch. Has some extra features

Extra Features

- Add delay in ms to tracks in bulk
- Rename tracks in bulk
